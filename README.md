# 🐾 Get A Pet

Projeto de adoção de pets desenvolvido com Node.js, React e MongoDB.

## 📋 Sobre o Projeto

A aplicação permite:

- Cadastro e login de usuários
- Edição de perfil
- Cadastro de pets
- Edição e remoção de pets
- Solicitação de adoção
- Gerenciamento do processo de adoção

A comunicação entre frontend e backend é feita através de uma API REST.

## 🚀 Tecnologias

### Backend

- Node.js
- Express
- MongoDB
- Mongoose
- JWT
- Bcrypt
- Zod

### Frontend

- React
- React Router DOM v6
- Axios
- CSS Modules

## ✨ Melhorias Implementadas

Além do projeto base, foram realizadas algumas melhorias:

- Validações utilizando Zod.
- Separação da lógica de negócio através da camada UserService.
- Atualização do React Router DOM da versão 5 para a versão 6.
- Melhor organização do código para facilitar manutenção.
